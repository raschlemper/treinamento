Introdução

Vamos começar nosso treinamento em Java?


Para começar a estudar este modulo é necessário que você instale na sua máquina os seguintes softwares:

  • Git Sistema para gerenciamento e versionamento de arquivos.
    https://git-scm.com/download/win
  • Tomcat 8.0 Container de aplicação para rodar o projeto (Server).
    https://tomcat.apache.org/download-80.cgi
  • Eclipse Neon Software para criação do projeto (IDE).
    https://www.eclipse.org/downloads/

Depois de você instalar esses software, nos podemos começar o desenvolvimento de uma aplicação simples para entender como funciona o fluxo de uma aplicação em java.


Objetivos iniciais:

  • Entender como funcionam as principais ferramentes de desenvolvimento em java.
  • Entender como funciona o fluxo de desenvolvimento de uma aplicação em java.
  • Criar nossa primeira aplicação em java, o famoso Hello world!!!

Uma boa maneira de estudar é após você instalar esses software, dar uma lida para entender do que se trata cada um deles.

Quanto mais rápido nos caminharmos com o aprendizado, mas rápido começarmos a criar nos aplicações.
